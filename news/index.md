# Changelog

## midnight (development version) 0.1.0.900-909

- Added wrapper method for `ggmid.mid.importance` with new types:
  “violin”, “beeswarm”, “sinaplot”.
- Renamed main functions (from `mid_surrogate()` to
  [`mid_reg()`](https://ryo-asashi.github.io/midnight/reference/mid_reg.md)).
- Added
  [`persp.mid()`](https://ryo-asashi.github.io/midnight/reference/persp.mid.md).
- Added the “moon” and “moonlit” color themes
- Added the package document.
- Updated wrapper fit function `fit_mid_mid()` to improve memory
  efficiency.
- Opened pkgdown website and added favicons.
- Added `mid_surrogate()` and `make_mid_mid()`.
- Created the package hex logo.
